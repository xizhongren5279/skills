# 比亚迪 公司一页纸

## 1. 公司近况

比亚迪2025年以460.2万辆新能源汽车销量登顶中国车企榜首，确立行业领导地位。2026年1月公司销量出现显著内外分化：国内销量承压主要因春节前经销商去库存、渠道调整及存量竞争白热化导致需求分流；海外出口持续亮眼，得益于土耳其、巴西、欧洲等地销量快速增长及自建滚装船运力提升，1月海外销量10万辆同比+50.8%、占比达47.8%成为销量新支柱。公司加速推进"技术平权"战略，2026年1月28日发布天神之眼5.0辅助驾驶系统，刀片电池等核心技术持续赋能产品矩阵，市场关注后续新车型周期及旺季销量回暖，当前股价处于与基本面共振上行的机遇窗口期。

## 2. 核心投资逻辑

### 2.1 短期逻辑

- **低库存+新技术开年，销量数据更扎实**：公司已大幅去库存，后续销量数据将更真实可靠；2026年重点布局低温快充、长续航、城区NOA普及等新技术，截至2025年底辅助驾驶车保有量256万辆、每天生成数据超1.6亿公里，规模优势驱动数据快速积累，天神之眼5.0辅助驾驶系统发布增强竞争力。

- **海外成盈利核心增长点**：1月海外销量10万辆、同比+50.8%、占比达47.8%成为销量新支柱；泰国、乌兹别克斯坦、巴西工厂已投产，匈牙利工厂将于2026Q2正式投产；若欧盟关税取消，单车利润可提升2-3倍，有效遏制海外价格战，欧洲销售点计划从1千个增长至2千个。

- **新产品周期蓄势待发**：2026年正式开启"大电池混插"时代，推出2026款秦PLUS DM-i、海豹05DM-i、秦L DM-i和海豹06DM-i四款新车型，ASP提升、电池扩容、功能优化同步推进；静待春节后旺季来临，随着各省份补贴陆续落地和年前购车需求释放，二三月份终端促销有望推动销量底部向上。

- **外部电池业务打开增长空间**：2026年外部电池销售可带来额外150亿元净利润（vs. 2025E），市场基本忽视比亚迪电池业务隐含价值，存在明显低估，作为全球第二大电池制造商在储能热潮中被市场忽视。

- **股价已充分反应悲观预期**：经历持续回调后，当前PE TTM为20.7倍，处于5年维度下的7.4%分位值，2026年保守利润预计约347-466亿元，估值具备吸引力，处于价值底部支撑区域。

### 2.2 长期逻辑

- **全产业链布局构筑技术护城河**：刀片电池等核心技术全面赋能产品矩阵，产品覆盖高中低端全市场；已实现电气化全产业链垂直整合，从电池、电机、电控到整车制造，技术创新型科技企业定位明确，累计研发投入超2200亿元，研发费用率维持7%+水平。

- **海外扩张提供长期增长确定性**：2026年海外销量目标150万辆（2025年104.6万辆），是中国新能源出海核心标的；本地化产能建设持续推进（泰国、巴西、匈牙利等工厂），海外市场覆盖东南亚、拉美、欧洲及大洋洲结构均衡，定价与管理体系成熟，未来十年海外市场将成为关键增长驱动力。

- **高质量增长转型龙头优势明显**：随着新能源渗透率增长放缓，市场由高速增长向高质量增长过渡；作为行业龙头，凭借规模优势和应变能力调节经营质量，高端化转型初见成效（腾势、方程豹、仰望三大高端品牌加速布局），ASP提升和毛利率改善持续推进。

- **盈利能力持续提升**：2025-2027年归母净利润CAGR=24.1%、PEG<1，具备估值吸引力；外部电池销售、海外高利润业务、技术平权战略协同发力，推动单车盈利和整体ROE提升，2024年ROE达24.84%。

- **智能化与平台技术革新**：超级e平台发布，纯电技术迎来重大革新，持续巩固电动化技术优势；2026年有望推动长续航混动、兆瓦闪充（3万转电机）等关键技术落地，提升用户体验和产品竞争力。

### 2.3 风险提示

- **经济下行冲击终端需求**：经济放缓可能导致汽车等大宗消费品终端需求萎缩，影响出货量预期并传导至电池需求；全球经济放缓或推迟电动汽车普及进程，消费者观望情绪可能延续影响短期销量。

- **国内存量竞争加剧**：新能源市场存量竞争白热化，竞品持续发力分流需求，1月国内销量承压反映竞争压力；补贴退坡和新能源购置税补贴政策变化可能影响终端需求。

- **海外政策不确定性**：海外扩张面临关税、贸易壁垒等政策风险，虽然欧盟关税可能取消利好，但政策细则尚待明确，不确定性仍存；本地化产能建设进度可能不及预期。

## 4. 业务拆分

### 4.1 公司盈利方式

- **新能源汽车全产业链销售**：通过垂直整合电池、电机、电控到整车制造全产业链，2025H1单车盈利0.9万元/辆、单车ASP约13.7万元，2025年销量460.2万辆登顶中国车企，规模优势强化零部件采购能力和价格竞争力。

- **外部电池销售业务**：全球第二大电池制造商，2026年外部电池销售可带来额外150亿元净利润，电池产能扩张为对外供应奠定基础，刀片电池等核心技术全面赋能，在储能热潮中被市场忽视存在价值重估空间。

- **手机及电子业务**：比亚迪电子收购Jabil移动电子制造业务拓宽智能手机零部件业务，折叠屏手机带来单机价值量提升，AI服务器业务提供潜在增长空间，手机业务规模增长显著。

### 4.2 市场格局

比亚迪2025年新能源汽车销量460.2万辆登顶中国车企榜首，纯电车销量225.7万辆超越特斯拉成为全球冠军，在0-10万元价格带市场份额占据第一、10-20万元价格带份额稳固第一、20-25万元价格带稳居第二，全价格带覆盖竞争优势明显。主要竞争对手包括特斯拉（全球纯电市场、PE 69.3倍）、吉利汽车（国内新能源市场、PE TTM 10倍）、理想汽车（高端新能源市场、PE 25.10倍）、小米汽车（智能电动车新势力）。

### 4.3 各板块业务情况

- **王朝&海洋品牌**：主力品牌覆盖10-20万元主流市场，1月销量17.8万辆，秦Plus DM-i、驱逐舰05、海豚等车型矩阵丰富，2026年推出"大电池混插"新车型（秦PLUS DM-i、海豹05DM-i等）。

- **高端品牌**：腾势（1月销量21,581辆）、方程豹（1月销量6,002辆、2025年全年23.50万辆同比+316.0%）、仰望（1月销量413辆）三大高端品牌加速布局，车型储备丰富，高端化转型初见成效。

- **电池及储能业务**：动力电池业务支撑新能源汽车销售，全球第二大电池制造商，2026年外部电池销售目标150亿元净利润，刀片电池技术优势明显。

- **电子业务**：手机零部件（折叠屏带来价值量提升）、AI服务器业务拓展，比亚迪电子通过收购Jabil业务扩大规模，为多元化盈利提供支撑。

## 5. 财务与估值快照

### 5.1 最新财报解读

- **收入高增、盈利稳健**：2024年营收7771亿元同比+29.02%，归母净利润416亿元同比+33.00%，净利率5.35%、ROE 24.84%，在行业竞争加剧背景下保持盈利能力稳健，2025H1单车盈利0.9万元/辆。

- **毛利率承压但环比改善**：综合毛利率17.01%（2024年）较2023年21.22%下降，主要因价格竞争及产品结构调整，但2025Q3单车盈利环比提升至0.67万元，盈利能力开始改善。

- **现金流稳定支撑扩张**：2024年经营现金流充裕，资产负债率74.64%（较2023年77.86%下降），财务结构优化，支撑海外产能建设和研发投入（研发费用率7%+），为全球化战略和技术创新提供财务安全垫。
