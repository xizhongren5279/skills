## 财务分析维度

### 1. 收入分析

**核心指标**
- 营业收入总额及增长趋势（YoY, QoQ）
- 收入结构：按产品线、按客户、按地区分布
- 收入质量：现金收入占比、应收账款周转率
- 收入集中度：前五大客户占比
- 收入真实性：与发票、银行流水的交叉验证

**关注要点**
- 收入确认政策是否合理
- 是否存在突击收入或收入提前确认
- 关联交易占比及定价公允性
- 季节性波动及其原因
- 退货率及其变化趋势

### 2. 成本与利润分析

**核心指标**
- 毛利率水平及变化趋势
- 营业利润率、净利率
- EBITDA 及 EBITDA 利润率
- 成本结构：原材料、人工、制造费用占比
- 期间费用率：销售、管理、财务费用

**关注要点**
- 毛利率是否符合行业水平
- 利润率波动的驱动因素
- 非经常性损益占比
- 成本控制能力
- 费用合理性（如高管薪酬、关联交易费用）

### 3. 现金流分析

**核心指标**
- 经营活动现金流净额
- 投资活动现金流净额
- 筹资活动现金流净额
- 自由现金流（FCF）= 经营现金流 - 资本支出
- 现金流量比率 = 经营现金流 / 流动负债

**关注要点**
- 经营现金流与净利润的匹配度
- 现金流为负的原因及可持续性
- 资本支出的合理性
- 现金流对外部融资的依赖程度

### 4. 资产负债表分析

**核心指标**
- 总资产及资产结构
- 总负债及负债结构
- 净资产及股东权益
- 资产负债率、流动比率、速动比率
- 应收账款周转天数、存货周转天数

**关注要点**
- 资产质量：应收账款账龄、存货积压、固定资产折旧
- 负债结构：短期负债压力、长期偿债能力
- 或有负债：担保、诉讼等
- 表外负债：融资租赁等
- 资产真实性验证

### 5. 财务比率分析

**盈利能力**
- ROA（资产回报率）= 净利润 / 平均总资产
- ROE（净资产收益率）= 净利润 / 平均股东权益
- ROIC（投入资本回报率）

**偿债能力**
- 流动比率 = 流动资产 / 流动负债
- 速动比率 = （流动资产 - 存货）/ 流动负债
- 利息保障倍数 = EBIT / 利息费用

**运营效率**
- 应收账款周转率 = 营业收入 / 平均应收账款
- 存货周转率 = 营业成本 / 平均存货
- 总资产周转率 = 营业收入 / 平均总资产

## 业务分析维度

### 1. 商业模式评估

**核心问题**
- 公司的核心产品/服务是什么？
- 价值主张：为客户解决什么问题？
- 收入模式：如何获得收入（订阅、交易、广告等）？
- 客户获取：如何触达和获取客户？
- 成本结构：主要成本驱动因素是什么？

**评估要点**
- 商业模式的可持续性和可扩展性
- 竞争壁垒：技术、品牌、网络效应、规模经济
- 对供应商和客户的议价能力
- 业务模式创新性及差异化

### 2. 市场与行业分析

**市场规模**
- TAM（总体可获得市场）、SAM（可服务市场）、SOM（可获得市场份额）
- 市场增长率及驱动因素
- 行业生命周期阶段

**竞争格局**
- 主要竞争对手及市场份额
- 公司市场定位（领导者、挑战者、跟随者、利基）
- 竞争优势分析（波特五力模型）
- 行业集中度及趋势

**市场趋势**
- 宏观经济环境影响
- 政策法规变化
- 技术发展趋势
- 消费者行为变化

### 3. 产品与技术

**产品组合**
- 核心产品及其市场表现
- 产品生命周期阶段
- 产品线丰富度及互补性
- 新产品研发管线

**技术能力**
- 核心技术及其先进性
- 研发投入占比及研发团队规模
- 专利及知识产权布局
- 技术迭代速度与行业比较

### 4. 客户与销售

**客户分析**
- 客户画像：行业、规模、地域分布
- 客户获取成本（CAC）
- 客户生命周期价值（LTV）
- LTV/CAC 比率（健康比率 > 3）
- 客户留存率及流失率

**销售渠道**
- 直销、分销、线上、线下等渠道占比
- 渠道效率及成本
- 渠道依赖风险

## 法律合规维度

### 1. 公司基本信息

**工商注册信息**
- 公司全称、注册地址、统一社会信用代码
- 注册资本及实缴情况
- 成立时间及经营期限
- 经营范围及业务资质

**股权结构**
- 股东名册及持股比例
- 实际控制人识别
- 股权代持、对赌协议等特殊安排
- 历史股权变动记录

### 2. 合规性审查

**资质与许可**
- 经营所需的行业资质、许可证
- 资质有效期及续期情况
- 特许经营权

**税务合规**
- 纳税申报记录
- 是否享受税收优惠政策
- 是否存在税务争议或处罚

**环保与安全**
- 环评批复及验收情况
- 排污许可证
- 安全生产许可
- 环保处罚记录

### 3. 法律风险

**诉讼与仲裁**
- 正在进行的诉讼/仲裁案件
- 历史诉讼记录及影响
- 潜在法律纠纷

**知识产权**
- 商标、专利、著作权清单
- 知识产权权属是否清晰
- 是否存在侵权风险
- 核心技术是否涉及职务发明纠纷

**重大合同**
- 重大销售/采购合同审查
- 借款合同及担保合同
- 租赁合同（办公场所、生产设备）
- 关联交易合同

### 4. 劳动与员工

**劳动合同**
- 劳动合同签订率
- 社保公积金缴纳情况
- 劳动争议及仲裁记录

**核心员工**
- 竞业限制协议
- 保密协议
- 股权激励安排

## 团队背景维度

### 1. 管理团队

**核心高管背景**
- 创始人/CEO：教育背景、从业经历、创业经验
- CFO：财务专业背景、资本市场经验
- CTO：技术能力、研发管理经验
- 其他核心高管：销售、运营等

**团队评估要点**
- 管理团队完整性及稳定性
- 行业经验及资源积累
- 成功案例及失败教训
- 团队协作及互补性
- 诚信记录（是否有失信、犯罪记录）

### 2. 股东背景

**主要股东**
- 自然人股东：个人背景、资金来源
- 机构股东：投资机构背景、投资偏好
- 战略股东：产业资源、协同效应

**股东尽调要点**
- 股东实力及信誉
- 股东之间的关系（一致行动人、关联方）
- 股东承诺及义务
- 潜在股东变动风险

### 3. 员工结构

**组织架构**
- 部门设置及职责划分
- 层级结构及汇报关系
- 决策机制

**人员分析**
- 员工总数及变化趋势
- 人员结构：按职能、按学历、按工龄
- 核心技术/业务人员占比
- 员工流失率及原因

## 风险评估清单

### 财务风险

- [ ] 收入确认政策不当或造假
- [ ] 利润严重依赖非经常性损益
- [ ] 经营现金流长期为负
- [ ] 应收账款坏账风险高
- [ ] 存货积压或贬值
- [ ] 过度依赖债务融资
- [ ] 短期偿债压力大
- [ ] 关联交易占比过高且定价不公允

### 业务风险

- [ ] 商业模式不清晰或不可持续
- [ ] 市场竞争激烈，缺乏差异化
- [ ] 客户集中度过高（前5大客户 > 50%）
- [ ] 供应商集中度过高
- [ ] 核心技术落后或面临淘汰
- [ ] 产品单一，抗风险能力弱
- [ ] 行业政策变化风险
- [ ] 宏观经济下行影响

### 法律合规风险

- [ ] 经营资质不全或即将到期
- [ ] 存在重大诉讼或仲裁
- [ ] 税务合规问题
- [ ] 环保违规记录
- [ ] 知识产权纠纷
- [ ] 劳动纠纷频发
- [ ] 重大合同存在瑕疵
- [ ] 实际控制人失信或犯罪记录

### 团队风险

- [ ] 核心管理层不完整
- [ ] 创始人/CEO 能力不足
- [ ] 管理团队稳定性差，频繁变动
- [ ] 股东之间存在重大分歧
- [ ] 核心技术人员流失严重
- [ ] 缺乏行业经验和资源
- [ ] 诚信问题或不良记录

## 数据验证方法

### 文档交叉验证

**财务数据验证**
- 财务报表 ↔ 审计报告
- 报表数据 ↔ 银行流水
- 收入数据 ↔ 发票、合同
- 成本数据 ↔ 采购合同、付款凭证
- 税务申报 ↔ 财务报表

**业务数据验证**
- 销售数据 ↔ 客户访谈
- 市场份额 ↔ 行业报告
- 产能数据 ↔ 实地考察
- 研发投入 ↔ 项目台账

### 外部信息验证

**公开信息查询**
- 工商登记信息（企查查、天眼查）
- 诉讼仲裁信息（中国裁判文书网）
- 失信被执行人（中国执行信息公开网）
- 行政处罚信息（信用中国）
- 专利商标查询（国家知识产权局）

**第三方调研**
- 客户访谈
- 供应商访谈
- 行业专家访谈
- 竞争对手调研
- 实地考察

### 数据合理性判断

**逻辑一致性检验**
- 收入与成本的匹配关系
- 利润与税负的匹配关系
- 资产增长与业务增长的匹配
- 行业对标分析

**异常数据标记**
- 突然大幅波动的指标
- 与行业平均严重偏离的数据
- 逻辑上相互矛盾的信息
- 口径前后不一致的说法

## 使用建议

1. **系统化分析**：按照四大维度（财务、业务、法律、团队）逐一展开，确保全面覆盖

2. **重点突出**：根据投资目的和行业特点，有侧重地深入分析关键风险点

3. **交叉验证**：不依赖单一数据源，通过多方信息交叉印证

4. **实地调研**：重要项目必须进行实地考察和访谈

5. **专业协作**：涉及法律、财务、技术等专业领域时，寻求专业机构协助

6. **动态跟踪**：尽调不是一次性工作，投资后应持续跟踪关键指标变化
